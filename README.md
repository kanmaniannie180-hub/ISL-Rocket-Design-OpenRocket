# 📖 Overview

Rocket design requires a careful balance between aerodynamics, stability, propulsion, and recovery mechanisms. Even small design modifications can significantly impact flight performance, altitude, velocity, and mission success.

This project presents the complete design and simulation of a **single-stage model rocket** using **OpenRocket**, a professional open-source rocket simulation platform widely used for educational and aerospace research purposes.

The study investigates how different motor configurations influence:

* 🚀 Maximum Altitude (Apogee)
* ⚡ Velocity Profiles
* 📈 Acceleration Characteristics
* 🛰️ Flight Stability
* 🪂 Recovery System Performance
* ⏱️ Flight Duration

Multiple simulation scenarios were performed to evaluate rocket behavior under varying propulsion systems and to determine the most effective configuration for achieving high-altitude, stable flight.
